Station Configuration

Each station is configured with specific hardware and software roles. All client stations read aircraft data from the server and synchronize in real-time to ensure a cohesive training environment.

Server & Instructor

  • PC: High-End with dedicated GPU
  • Displays: 1x Large TV, 2x Monitors
  • Peripherals: Keyboard, Mouse, Headset
  • Manages the core simulation state
  • Provides master ASR/PAR overview
  • Has master control over the simulation

Officer/Pilot

  • PC: Mid-Range
  • Displays: 1x Monitor
  • Peripherals: Keyboard, Mouse, Headset
  • Receives real-time simulation data
  • Full interactive ASR/PAR view
  • PTT voice for pilot communication

Student/ATC

  • PC: Mid-Range
  • Displays: 1x Monitor
  • Peripherals: Keyboard, Mouse, Headset
  • Receives real-time simulation data
  • Read-only ASR/PAR view
  • PTT voice for ATC control

Network & Cable Management

A clean and organized setup is crucial for maintenance and reliability.

  • Network: All computers will be connected via a dedicated, high-speed Local Area Network (LAN) using the central network switch to ensure low latency.
  • Cabling: All power and network cables will be managed using under-desk cable trays and raceways to prevent clutter and tripping hazards.
  • Peripherals: Each station will be equipped with a monitor, keyboard, mouse, and headset, with cables neatly routed.
