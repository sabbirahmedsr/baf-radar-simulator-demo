Command Help

Format: <Callsign> <Verb> <Param>

H <heading>
Set heading (0-359). Ex: H 270

A <altitude>
Set altitude (x1000 ft). Ex: A 9 (for 9,000 ft)

S <speed>
Set speed in knots. Ex: S 250

Commands can be chained:

AC101 H 270 A 8 S 210
